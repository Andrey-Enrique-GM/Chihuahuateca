🐾 Sobre el Proyecto

¿Qué es la "Chihuahuateca"?

"Chihuahuateca" es una aplicación web funcional y minimalista orientada a ser el guardián ideal de tus colecciones personales de libros, películas y series.

Tecnologías Utilizadas

Python Flask MySql HTML5 & CSS3 JavaScript

Servicios e Infraestructura

Para asegurar un rendimiento fluido, gratuito y en la nube, el sistema está integrado con los siguientes servicios externos:

  • Base de Datos en la nube: Hospedada de forma remota en la nube gratuita de Aiven MySQL.
  • Despliegue y Hosting: Servidores web administrados a través de la plataforma Render.
